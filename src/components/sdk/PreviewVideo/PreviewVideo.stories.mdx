# PreviewVideo

The `PreviewVideo` component renders a preview of the local user's video. If there is no available video input, the video will be blank. 

A `PreviewVideo` is typically used on a pre-meeting view, or a device settings page. The React SDK depends on the Amazon Chime SDK for JavaScript, a single video stream is attached to a video element at a time. Hence, if enabled preview video in the in-meeting screen page, closing preview video also leads to the clos of local video. We recommed using `PreviewVideo` component prior to in-meeting page.

## Importing

```javascript
import { PreviewVideo } from 'amazon-chime-sdk-component-library-react';
```

## Usage

```jsx
import React from 'react';
import {
  MeetingProvider,
  PreviewVideo
} from 'amazon-chime-sdk-component-library-react';

const App = () => {
  return (
    <MeetingProvider>
      <PreviewVideo />
    </MeetingProvider>
  );
};
```
